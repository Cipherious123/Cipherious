Build Your Own Cipher

Your password for your cipher must satisfy the following:
It must be 95 characters long
It must contain all of the following 95 letters in any order, with no repetition
ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789+/!"#$%&'()*,-.:;<=>?@[\\]^_`{|}~space

The first letter in your password that all characters 'A' in the plaintext should be replaced by that letter
The second letter does the same for 'B', and so on till ' '(space)
If you want to input 'space', use the spacebar, don't type space.
