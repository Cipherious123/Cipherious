Guidelines to Create a Combination

When using integer base change in a combination, input is always in base 95
Password must have: base to convert to, system of number, system to convert number to, all separated by spaces
Unicode must be conveyed as u, Alpha as a, Normal as n
For example, to convert to base 40 and normal to unicode, you write: '40 n u'

Substitution, Caesar and BYOC work best with smaller texts as they are easy to find patterns in for larger texts
Scrambler works best for larger texts as it only jumbles up the letters.
If you leave any password blank empty, the password for that step is randomly generated
A password can't be longer than 100 characters and a combo name can't be longer than 25.
All characters must be in basic latin unicode(alphabets, numbers, spacebar, punctuation)

Always save the name of the combination before you add its steps
Make sure there was no error after submitting a step
Don't forget to complete combination when you are done
